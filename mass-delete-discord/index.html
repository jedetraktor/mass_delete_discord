Mass-delete sur Discord

Attention cette manip ne permet de supprimer que les messages salon par salon. Cependant elle est beaucoup plus rapide que de le faire un par un

Cette manip peut paraître compliquée mais en réalité elle est très simple une fois qu'on a compris le principe.

Il s'agit juste de :

    • Ouvrir la console de commande
      Coller du code dedans
      Retrouver le token d'autorisation en 10sec
      Et enfin appuyer sur entrée ou exécuter

    • Etape 1 : Se connecter à Discord via votre navigateur Web

    La manip ne marche pas si vous le faites via l'appli

    Etape 2 : Se rendre sur le MP ou salon duquel vous souhaitez effacer l'intégralité de vos messages

    Si vous souhaitez effacer l'intégralité de vos messages d'un serveur entier (par exemple sur tout le Backup), il faudra selectionner salon après salon

    Etape 3 : La console de développement

    Cela va vous ouvrir le console de developpement. Il suffira de copier/coller une grosse part de code tout préparé dans la console

    Ouvrir la console de développement

    Sur Windows :

    • Chrome / Firefox (Windows, Linux, Chrome OS): F12 ou Controle+Shift+J

    • Sur Mac :

    • Chrome (Mac): Commande+Option+J
      Safari (Mac): Commande+Option+C

    Si la console de commande ne s'ouvre pas sur Safari : Allez dans le menu Safari / Réglages / Avancé et cochez tout en bas "Affichez les fonctionnalités pour les développeurs web"

    Etape 4 : Copier-coller le code dans la console de commande

    Ouvrez la page Accédez au code et copiez l'intégralité du code que vous collerez dans le rectangle de la console de commande.

    Sur PC, il me semble qu'il suffit de le coller en dessous du message d'avertissement.

    Sur Mac il se peut que lorsque vous ouvrez la console, vous arrivez sur l'inspecteur. Il suffit de cliquer sur le mot "console" dans la barre rectangulaire en haut et vous trouverez le message d'avertissement à droite. Vous pouvez donc coller le code à gauche.

    Ensuite appuye sur Entrée ou cliquer sur Exécuter

    Il se peut que vous ayez un message de blocage de pop-up. Si c'est le cas cliquez sur "Autoriser".

    Cela devrait donc s'ouvrir :

    Etape 5 : Recupérer les tokens

    Pour les token "Author" et "Guild/Channel" il suffira d'appuyer sur les boutons bleus "Auto". Cela devrait vous afficher dans les encarts, une série de chiffre

    Pour le token "Authorization", cliquer n'affiche plus les chiffres automatiquement. Il va falloir fouiller dans la console pour trouver cette clé d'autorisation

    Etape 6 : Trouver la clé d'autorisation

    <>
      Dans la console de commande, aller dans "Network" ou "Réseau"
      Allez sur "XHR"
      Tapez "api/v9" pour le filtre de recherches
      Dans la partie ou "En-têtes" ou "Headers" cherchez "En-tête de la requête". Vous devriez trouver "Authorization" suivi d'une longue suite de lettres et de chiffres. C'est votre token "Authorization"
      Copiez-le. Ensuite retournez sur la console. Refaites exécuter ou Entrée le code précédement collé et vous retomberez sur la fenêtre des tokens. Vous pourez alors entrer le token d'autorisation dans la fenêtre appropriée.


    NE PARTAGEZ JAMAIS VOTRE authTOKEN !

    Etape 7 : Cliquez sur le bouton "Start"

    Une petite fenêtre devrait s'ouvrir alors vous demandant si vous êtes sûr.e de vouloir supprimer X messages (selon votre nombre de message posté dans le MP ou le salon). Il suffira de cliquer sur "Ok" et les messages se supprimeront ensuite, le temps de suppression peut être parfois un peu long selon voytr nombre de messages postés dans le salon. Si vous souhaitez supprimer les autres messages des autres salons, il suffira de cliquer sur chaque salon et de recommencer la manip à partir de "Excuter" Bien entendu, vous n'aurez pas à retrouver le token d'authorization, dès lors que vous l'avez copié il suffira de le coller à nouveau au même endroit.
